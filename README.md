# Beginner C / C++ Problems

This repository contains beginner-level programming problems implemented in **C and C++**.

These programs are commonly used in **introductory programming courses and labs**.

## Topics Covered

### Basic Syntax

* Hello World
* Add two numbers
* Swap variables
* Quotient and remainder

### Control Flow

* Even or Odd
* Largest of three numbers
* Leap year checker
* Quadratic equation roots
* Simple calculator (switch)

### Loops and Math

* Fibonacci series
* Factorial
* Prime number check
* Reverse number
* Palindrome number

### Arrays and Strings

* Largest and smallest element in array
* Average of array
* Reverse array
* String length without `strlen()`
* String palindrome

## Languages

* C
* C++

## Purpose

This repository is intended for **learning basic programming logic and syntax**.
